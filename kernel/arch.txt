Цель:
    - создание "внутренней" Безопасной ОС (БОС) (Secure OS)
    - создание ядра "внутренней" ОС, используемой в разрабатываемом оборудовании (ядро БОС)
    - порт для процессора imx6q6 (ARM Cortex-A9)


Требования:
    - БЕЗОПАСНОСТЬ
    - НАДЕЖНОСТЬ
    - аппаратная защита памяти (MMU)
    - поддержка аппаратной многоядерности (SMP)
            
        
Для разработчика (рекомендации):
    - используйте единый стиль кодирования
    - старайтесь избегать хиростей! (KISS)
    - читабельность ОЧЕНЬ важна
    - документация в виде комментирования модулей и API
        - в .h файлах описание назначения модуля
        - в .c файлах описание реализации


Архитектура ОС:
    - ОСРВ (мягкое время)
        - Поскольку системное и прикладное ПО создется только собственными силами, 
          и основное планируемое применение: для изделий, то выбрана модель ОС РВ.
    - микроядро
        - надежность и модульность
        - интерфейс программирования: набор системных вызовов (см. описание)
